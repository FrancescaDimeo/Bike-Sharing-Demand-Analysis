# Global Bike-Sharing Demand Prediction

# Project Overview
This project focuses on predicting bike-sharing demand in major urban centers by analyzing meteorological and temporal factors. It was developed as the final **Capstone Project** for the **IBM Applied Data Science with R** Specialization.

# Technical Stack
- **Language:** R
- **Database Management:** SQL
- **Key Libraries:** `tidyverse`, `ggplot2`, `tidymodels`, `leaflet`, `Shiny`
- **Tools:** Jupyter Notebook, IBM DB2

# Key Achievements
- **Data Pipeline:** Integrated multiple data sources using **Web Scraping** (Wikipedia) and **API calls** (OpenWeather).
- **Predictive Modeling:** Built and optimized Linear and Polynomial Regression models to forecast rental counts.
- **Model Performance:** Achieved an **R-squared of 0.75**, demonstrating high predictive power.
- **Geospatial Visualization:** Created interactive maps and dashboards to identify high-demand "hotspots" in Seoul and New York.

# Repository Content
- `bike_sharing_analysis.R`: Full R script covering data cleaning, EDA, and modeling.
- `Final_Project_Report.pdf`: Comprehensive business presentation with insights and visualizations.
